# Node Chat App (Demo)

This is a demo **Node.js + Express + MongoDB** project built by **Gopal Maddheshiya**.  
It is a simple chat application where users’ messages are stored in MongoDB using Mongoose.

## Features
- Node.js + Express backend  
- MongoDB database connection via Mongoose  
- EJS templates for dynamic frontend  
- Method Override for PUT & DELETE routes  
- Clean MVC folder structure  
- Static assets served via Express  
- Demo-friendly configuration for learning purposes

## Setup (Run Locally)

Follow these steps to run this project on your local system 

```bash
# 1️⃣ Clone this repository
git clone https://github.com/Gopal196216/node-chat-api1.git

# 2️⃣ Navigate to the project folder
cd node-chat-api1

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run the app
node index.js

Now open your browser and visit:

http://localhost:8080/
